có 3 cách để thêm 1 css
1. Inline css: thêm trực tiếp vào thẻ html
2. Internal css: thêm vào trong thẻ style trong phần head
3. External css: thêm vào file css riêng và liên kết với file html bằng thẻ link
-với cách 3, ta có thể dùng lại file css cho nhiều file html khác nhau, giúp tiết kiệm thời gian và công sức
-với cách 2, ta có thể viết css nhanh chóng mà không cần tạo file css riêng
-với cách 1, ta có thể viết css nhanh chóng cho từng thẻ html cụ thể
-cú pháp của cách 1: thẻ html style="thuộc tính: giá trị;"
-cú pháp của cách 2: thẻ style trong phần head, bên trong thẻ style là các quy tắc css
-cú pháp của cách 3: thẻ link trong phần head, với thuộc tính rel="stylesheet" và href="tên file css"
