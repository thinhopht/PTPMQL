<head>: Là nơi chứa siêu dữ liệu về tài liệu HTML.

<title>: Xác định tiêu đề của trang (hiện trên tab).

<meta>: Cung cấp metadata như mô tả, từ khóa, tác giả, charset.

<base>: Đặt URL gốc cho các liên kết tương đối trong trang.

<link>: Liên kết tới file CSS hoặc tài nguyên bên ngoài.

<style>: Viết CSS trực tiếp trong file HTML.

<script>: Viết JavaScript để thêm chức năng cho trang.
