dưới đây là danh sách các thuộc tính hay dùng trong form mà cần phải nhớ:
type → loại input (text, email, password, number, …)

name → tên biến khi gửi form (server nhận dữ liệu theo name)

value → giá trị mặc định

placeholder → gợi ý trong ô nhập

required → bắt buộc nhập

readonly → chỉ đọc, không sửa được

disabled → bị vô hiệu hóa

min, max → giới hạn số hoặc ngày tháng

minlength, maxlength → giới hạn số ký tự

pattern → định dạng theo regex

checked → mặc định được chọn (radio/checkbox)

multiple → cho phép chọn nhiều (file, email)

autofocus → focus sẵn khi mở form
